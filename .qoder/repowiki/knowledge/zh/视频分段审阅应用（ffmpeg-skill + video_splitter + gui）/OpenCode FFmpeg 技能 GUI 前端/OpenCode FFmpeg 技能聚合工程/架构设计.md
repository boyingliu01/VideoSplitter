本仓库是 OpenCode Skills 生态中的一个聚合工程，围绕 `ffmpeg-skill/`（底层封装）→ `video_splitter/`（业务编排）→ `gui/`（桌面入口）的单向依赖链组织：
- `ffmpeg-skill/` 作为独立 pip 包与 `.opencode/skills/ffmpeg-video/SKILL.md` 双形态发布，被上层消费；`video_splitter/` 仅允许 import `ffmpeg_skill`，禁止反向引用。
- `pyproject.toml` + `requirements.txt` 声明 Python ≥3.12 及 faster-whisper、PySide6、funasr、torch、openai 等全部第三方依赖，所有子模块共享同一虚拟环境。
- `architecture.yaml` 由 xp-gate 质量门禁读取，强制分层白名单，确保 ffmpeg 封装与业务编排解耦。
- `install.sh` / `install.bat` 是跨平台入口：检测 Python/FFmpeg → `pip install -r requirements.txt` → 将 `ffmpeg_tool.py` 软链到 `$HOME/.local/bin/ffmpeg-tool` → 注册 OpenCode skill，一次脚本启动全部子能力。
- 测试与评测集中在 `tests/` 与 `ffmpeg-skill/tests.py`，由 `[tool.pytest]` 统一 discover，并受 coverage 50% 门槛约束。
