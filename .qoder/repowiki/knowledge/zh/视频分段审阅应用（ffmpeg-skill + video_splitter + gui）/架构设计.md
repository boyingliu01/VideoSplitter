根目录通过 architecture.yaml 声明三层依赖边界：ffmpeg-skill 仅自引用；video_splitter 可导入 ffmpeg_skill；gui 可导入 video_splitter 与 ffmpeg_skill，形成单向依赖链，由 .xp-gate 质量门禁强制校验。运行时由 pyproject.toml 统一注册 package 名 video_splitter，pytest 同时扫描 tests/ 与 video_splitter/tests/，coverage 覆盖 video_splitter 与 gui 两个子包，从而把 CLI 流水线、GUI 控制器/工作线程与 FFmpeg 技能聚合在同一个 Python 工程内，通过共享的 ffmpeg_skill 抽象屏蔽外部进程差异。
