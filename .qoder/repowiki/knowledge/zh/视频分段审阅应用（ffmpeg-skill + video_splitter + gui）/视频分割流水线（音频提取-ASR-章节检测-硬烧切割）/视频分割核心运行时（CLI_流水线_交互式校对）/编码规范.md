- 所有对外暴露的 CLI 子命令统一采用 `cmd_<name>(args)` 函数 + `parser.add_parser(...).set_defaults(func=...)` 模式注册。
- 配置一律通过 `SplitConfig.from_env()` 获取，再由参数覆盖（如 `--model`、`--cut-mode`、`--max-duration`），不在模块内直接读环境变量。
- 长耗时步骤以 JSON 中间文件（`.transcript.json`、`.chapters.json`、`.review_progress.json`）作为可恢复检查点，配合 `resume` 标志跳过已完成的阶段。
- 对关键写操作采用原子保存：`save_transcript_atomic` 使用 `tempfile.mkstemp` + `os.replace`，防止中断导致损坏。
- 可选重型依赖（`faster_whisper`、`gui.app`、`to_srt`）在需要时局部 `import`，避免启动期硬依赖失败。
