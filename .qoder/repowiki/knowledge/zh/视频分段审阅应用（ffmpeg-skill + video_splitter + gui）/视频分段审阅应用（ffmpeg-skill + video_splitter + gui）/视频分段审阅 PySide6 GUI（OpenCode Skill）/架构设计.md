顶层 `app.py` 中的 `MainWindow` 作为 Qt 应用入口，通过 QSplitter 组合左侧 VideoPlayerWidget + 右侧 QTabWidget（SubtitlePanel + Timeline），并在菜单栏中挂载 ReviewController/SplitController；`controllers/` 下的 ReviewController 与 SplitController 是纯业务 QObject，维护当前片段索引与已修改集合，封装 transcript/progress 的加载、保存与 SRT 导出，通过 `segment_changed`/`error` 等 Signal 与 UI 解耦；`widgets/` 下自定义 QWidget 仅负责展示与用户输入，不持有业务状态；`workers/` 将耗时 ASR 调用封装为可在 QThread 中运行的 QObject，通过 progress/finished/error 信号回传结果避免阻塞主线程。依赖方向单向：MainWindow → Controller + Worker + Widgets；Controller 仅依赖 `video_splitter.review` 与 `video_splitter.extractor.transcribe`，不反向引用 GUI。整个包同时作为 OpenCode Skill 被安装脚本聚合构建。
