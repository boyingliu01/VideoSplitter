- 每个高层 API 方法在执行前先检查输入文件是否存在，不存在则抛出 `FileNotFoundError`，再组装参数列表并委托给 `_run_command`。
- 所有对外抛出的异常统一归约为自定义 `FFmpegError`，由上层 CLI 捕获后以 `✗` 前缀打印到 stderr 并返回非零退出码。
- 可选进度回调采用 `Optional[Callable[[float], None]]` 形式附加在每个处理方法签名末尾，通过 `_run_command` 从 ffmpeg stderr 的 `time=` 行触发。
- 常用枚举值（分辨率预设、视频/音频编解码器、编码 preset）集中在类级字典常量（`RESOLUTION_PRESETS`、`VIDEO_CODECS`、`AUDIO_CODECS`）中，供方法做参数校验。
- 时间字符串解析复用私有 `_parse_time` 方法，同时支持 `HH:MM:SS`、`MM:SS` 与纯秒数三种格式。
